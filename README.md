# Task Management System

## Project Overview
A comprehensive web-based task management application built with Flask, designed to help users organize, track, and manage their tasks efficiently.
Features

### Core Functionality

User Authentication
- Registration
- Login/Logout
- Password reset

Task Management

- Create tasks
- Update tasks
- Delete tasks
- View task details


### Advanced Features

Task Categorization

- Create custom categories
- Assign categories to tasks

Priority Levels

- Set task priorities
- Filter tasks by priority

Status Tracking

- Not Started
- In Progress
- Completed
- Blocked
